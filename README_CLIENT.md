# Client (React)
## Quick start
1. Install dependencies:
   ```
   npm install
   ```
2. Create `.env` file with `REACT_APP_API_URL` (pointing to your Render backend) and `REACT_APP_MY_USER_ID` (for demo).
3. Start dev server:
   ```
   npm start
   ```
4. Build for static site deployment:
   ```
   npm run build
   ```
